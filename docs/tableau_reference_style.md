# Tableau Reference Style

Reference dashboard:

https://public.tableau.com/app/profile/israel.ayo/viz/SuperstoreSalesDashboard_17403964185480/KPIsSummary

## Observed Style Direction

- Dark executive header with white text and a compact navigation bar.
- Active dashboard tab highlighted with a saturated blue underline.
- White KPI cards layered over the dark header and light dashboard canvas.
- Minimal card borders, subtle shadows, and sharp low-radius corners.
- Accent palette built around bright blue, purple, cyan, green, and red.
- KPI cards include a pale-blue icon block, a large metric value, a small month-over-month indicator, and compact spark bars.
- Main analytical area uses a large trend chart, a right-side detail table, and lower supporting cards.
- Visual density is high, but labels remain sparse and business-focused.

## Applied To This Project

The Olist dashboard preview follows the same visual system while using marketplace-specific content:

- GMV, AOV, total orders, and customers as the top KPI cards.
- Monthly GMV and order trend as the primary line chart.
- Product category GMV as the donut and horizontal bar analysis.
- Seller risk breakdown as the right-side detail table.
- Customer geography and composite category risk as lower operational panels.
